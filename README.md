## Case Studies

### 1. Checkout Process Requirements & Analysis
**Scenario:** An e-commerce platform experiencing a broken checkout flow — unresponsive checkout button and cart display failures causing a 20% increase in cart abandonment.

**What I delivered:**
- Current state analysis and problem statement
- Future state process design and flow diagram
- Functional and non-functional requirements (FR/NFR)
- Acceptance criteria (Given/When/Then format)
- User stories and business rules
- Requirements traceability matrix

### 2. HRIS Onboarding Workflow — IT Access & Equipment Provisioning
**Scenario:** An HRIS onboarding process relying on manual handoffs between HR, hiring managers, and IT — causing new hires to start Day 1 without required access or equipment.

**What I delivered:**
- Stakeholder analysis with influence and pain point mapping
- Current state analysis and root cause identification
- Future state redesign with automated workflow triggers
- Functional and non-functional requirements (FR/NFR)
- Acceptance criteria (Given/When/Then format)
- User stories, business rules, and role-based approval controls
- Day 1 Ready status model and success metrics
